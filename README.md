# 🍕 Pizza Sales Analysis Dashboard (MySQL + Power BI)

## 📌 Project Overview

This project analyzes pizza sales data using **MySQL** for data querying and **Power BI** for interactive dashboard creation. The objective is to identify sales trends, customer ordering patterns, best-selling pizzas, and business performance metrics that can help improve decision-making.

## 🎯 Project Objectives

* Analyze overall sales performance.
* Identify top and bottom-performing pizzas.
* Understand customer ordering behavior.
* Evaluate sales distribution by pizza category and size.
* Create an interactive dashboard for business insights.

## 🛠️ Tools & Technologies

* **MySQL** – Data storage and analysis
* **Power BI** – Data visualization and dashboard development
* **SQL** – Data extraction and KPI calculations

## 📊 Key Performance Indicators (KPIs)

* Total Revenue
* Average Order Value
* Total Pizzas Sold
* Total Orders
* Average Pizzas per Order

## 🔍 Business Questions Answered

* Which pizzas generate the highest revenue?
* Which pizzas have the lowest sales performance?
* What are the most popular pizza categories?
* How does revenue vary by pizza size?
* What are the peak ordering periods?

## 📈 Dashboard Features

* Revenue Overview
* Sales Trend Analysis
* Category-wise Sales Distribution
* Size-wise Sales Distribution
* Top 5 Pizzas by Revenue
* Bottom 5 Pizzas by Revenue
* Top 5 Pizzas by Quantity Sold
* Top 5 Pizzas by Orders

## 🗄️ SQL Analysis Performed

* Revenue Calculation
* Average Order Value Analysis
* Sales Percentage by Category
* Sales Percentage by Pizza Size
* Top & Bottom Performing Products
* Order Volume Analysis

## 📋 Key Insights

* Identified the highest revenue-generating pizzas.
* Analyzed customer preferences based on pizza categories and sizes.
* Discovered best-selling and least-selling products.
* Measured overall business performance through KPIs.

## 🚀 Outcome

This project demonstrates how **MySQL** and **Power BI** can be combined to transform raw sales data into meaningful business insights through data analysis and visualization.
